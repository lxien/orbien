# 端口池配置

端口池用于预先定义 TCP / UDP 隧道可分配的公网端口范围，限制自动分配时的选择范围，避免与系统中其他服务占用的端口冲突。

同协议下可声明多段，系统会将所有区间合并后作为该协议的可用端口池。

```toml
[[port_pool.tcp]]
start = 9050
end = 9099

[[port_pool.tcp]]
single = 10000

[[port_pool.udp]]
start = 9050
end = 9099
```

每条 `[[port_pool.tcp]]` 或 `[[port_pool.udp]]` 声明一段端口，支持两种写法（二选一，不可混用）：

| 写法  | 示例                            | 说明                     |
|:----|:------------------------------|:-----------------------|
| 区间  | `start = 9050` + `end = 9099` | `start` 与 `end` 必须成对配置 |
| 单端口 | `single = 10000`              | 等价于 `[10000, 10000]`   |

:::tip 提示
如果服务端部署在云服务器，还需要在云服务器的**安全组**中开放对应的端口范围。
:::
