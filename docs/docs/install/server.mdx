# 服务端

## 一、Docker（推荐）
通过`Docker`可以快速部署`orbiens`服务端，无需手动安装`Java` 运行环境。

### 1.1、一键安装

在具备`docker`环境的`Linux`云服务器中执行如下命令：

```shell
curl -fsSL https://raw.githubusercontent.com/lxien/orbien/main/scripts/install.sh -o install.sh && chmod +x install.sh && sudo sh install.sh
```

云服务器`安全组`放行如下端口：

- 8080：HTTP隧道代理端口
- 8020：管理面板端口
- 9527：服务端和客户端通信端口
- 9050-9099：TCP隧道代理端口

管理面板地址：`http://server-ip:8020` (admin:123456)

### 1.2、自定义安装

#### 创建 Docker 网络

创建独立网络供`orbiens`与`MySQL`通信：

```shell
docker network create orbiens-net
```

#### 启动MySQL
```shell
docker run -d \
  --name orbiens-mysql \
  --network orbiens-net \
  --restart unless-stopped \
  -e MYSQL_ROOT_PASSWORD=123456 \
  -e MYSQL_DATABASE=orbiens \
  -v /opt/orbiens/mysql/data:/var/lib/mysql \
  mysql:8.4
```

查看`MySQL`状态：

```shell
docker logs -f orbiens-mysql
```

#### 1.2.4、启动 orbiens
```shell
docker run -d \
  --name orbiens \
  --network orbiens-net \
  --restart unless-stopped \
  -p 8080:8080 \
  -p 8020:8020 \
  -p 9527:9527 \
  -p 9050-9099:9050-9099 \
  -e MYSQL_HOST=orbiens-mysql \
  -e MYSQL_DATABASE=orbiens \
  -e MYSQL_PASSWORD=123456 \
  -e JAVA_OPTS="-Xms256m -Xmx256m -XX:MaxDirectMemorySize=512m -XX:+UseG1GC" \
  -e TZ=Asia/Shanghai \
  -v /opt/orbiens/logs:/app/logs \
  lxien/orbiens:0.10.0
```

查看`orbiens`运行日志：

```shell
docker logs -f orbiens
```
